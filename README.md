# Bootstrapped with [Create React App]

### Tech Stacks

- axios
- formik
- yup
- tailwindcss
- postcss
- autoprefixer

### How to run the code

- Create a .env file with REACT_APP_BASE_URL defined
- Run npm install
- Run npm run start (Runs the app in the development mode)
- Run npm run build (Builds the app for production)
